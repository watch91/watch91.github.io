# 汉字加密解密工具使用说明
1. 选择模式：点击「加密模式」或「解密模式」切换对应功能；
2. 输入文本：在输入框中填写需要加密的原始汉字，或需要解密的密文；
3. 执行操作：点击「开始加密/开始解密」，结果将自动显示在输出框中；
4. 清空内容：点击「清空内容」可一键重置输入、输出框文本。

# Instructions for Chinese Character Encryption & Decryption Tool
1. Select Mode: Click **Encryption Mode** or **Decryption Mode** to switch functions.
2. Enter Text: Input the original Chinese characters to encrypt or the ciphertext to decrypt in the input box.
3. Execute Operation: Click **Start Encryption/Start Decryption**; the result will display automatically in the output box.
4. Clear Content: Click **Clear Content** to reset the text in the input and output boxes with one click.

If you want to use this program online, please visit "https://watch91.github.io/%E6%B1%89%E8%AF%AD%E5%8A%A0%E5%AF%86v4.0.html".

# 漢字暗号化・復号化ツール使用方法
1. モード選択：「暗号化モード」または「復号化モード」をクリックして機能を切り替えます。
2. テキスト入力：入力ボックスに暗号化する元の漢字、または復号化する暗号文を入力します。
3. 操作実行：「暗号化開始/復号化開始」をクリックすると、結果が出力ボックスに自動的に表示されます。
4. コンテンツ消去：「コンテンツ消去」をクリックすると、入力ボックスと出力ボックスのテキストが一括でリセットされます。

このプログラムをオンラインで使用したい場合、「https://watch91.github.io/%E6%B1%89%E8%AF%AD%E5%8A%A0%E5%AF%86v4.0.html」にアクセスしてください。

# Инструкция по использованию инструмента для шифрования и расшифровки китайских иероглифов
1. Выберите режим: Нажмите **Режим шифрования** или **Режим расшифровки**, чтобы переключить функции.
2. Введите текст: Введите исходные китайские иероглифы для шифрования или шифртекст для расшифровки в поле ввода.
3. Выполните операцию: Нажмите **Начать шифровку/Начать расшифровку**; результат автоматически отобразится в поле вывода.
4. Очистите контент: Нажмите **Очистить контент**, чтобы сбросить текст в полях ввода и вывода одним кликом.
